This file reports the changes between template versions.

*v2.0.10 - 09/07/2026
- Changed numeric citations to use compact grouped brackets, so multi-key citations such as \cite{key1,key2} render as [1,2] rather than [1], [2].

*v2.0.9 - 09/07/2026
- Made symmetrical 2.5cm side margins the default layout and replaced the title-page UCL logo with a vector asset generated from the official UCL primary SVG.

*v2.0.8 - 09/07/2026
- Set major thesis titles, running headers, and page numbers in the UCL deep purple, and removed the running header rule for a cleaner navigation layer.

*v2.0.7 - 08/07/2026
- Unified the visual colour system around black text, muted navigation, and restrained UCL purple accents; added the symmetricmargins class option for electronic drafts.

*v2.0.6 - 08/07/2026
- Adjusted the book-like style toward a single-column SIGGRAPH-inspired layout with wider readable measure, Libertinus fonts when available, upright headers, sans structural headings, and title-case title page text.

*v2.0.5 - 08/07/2026
- Introduced a quieter book-like thesis style with narrower text block, reduced paragraph spacing, lighter headers, hidden hyperlink boxes, and more restrained chapter openings.

*v2.0.4 - 08/07/2026
- Changed the default thesis body spacing from double spacing to one-and-a-half spacing.

*v2.0.3 - 08/07/2026
- Added a standalone PDF preview file for the GitHub repository.

*v2.0.2 - 08/07/2026
- Generalised the template placeholder text by removing project-specific example content.
- Replaced specific title-page metadata with neutral placeholders.

*v2.0.1 - 08/07/2026
- Added a thesis-level literature review chapter between the introduction and the paper-based chapters.

*v2.0.0 - 07/07/2026
- Adapted the original report template into a UCL PhD thesis template.
- Replaced the title-page logo with the current UCL brand logo image.
- Updated page headers, chapter titles, and edge labels to use the UCL purple palette.
- Simplified the setup section for PhD thesis metadata.
- Replaced institution-specific copyright guidance with a UCL submission placeholder.
- Added a thesis-by-publication structure with a publications and author contributions page.
- Added four paper-based chapter templates and a reusable chapter publication note.
- Added Overleaf latexmkrc and pdfLaTeX fallback package loading to avoid fontspec engine errors.
- Changed the default book layout to oneside/openany to avoid automatically inserted blank pages in the electronic thesis version.
